HTML-теги

HTML-теги — основа языка HTML. Теги используются для разграничения начала и конца элементов в разметке. Каждый HTML-документ состоит из дерева HTML-элементов и текста. Каждый HTML-элемент обозначается начальным (открывающим) и конечным (закрывающим) тегом. Открывающий и закрывающий теги содержат имя тега.

HTML-атрибуты

HTML-атрибуты сообщают браузеру, каким образом должен отображаться тот или иной элемент страницы. Атрибуты позволяют сделать более разнообразными внешний вид информации, добавляемой с помощью одинаковых тегов. Значение атрибута заключается в кавычки "". Названия и значения атрибутов не чувствительны к регистру, но, тем не менее, рекомендуется набирать их в нижнем регистре.

HTML-ссылки

HTML-ссылки создаются с помощью элементов < a >, < area > и < link >. Ссылки представляют собой связь между двумя ресурсами, одним из которых является текущий документ. ссылки на внешние ресурсы — создаются с помощью элемента < link > и используются для расширения возможностей текущего документа при обработке браузером; гиперссылки — ссылки на другие ресурсы, которые пользователь может посетить или загрузить.
